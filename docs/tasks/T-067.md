# T-067: Show help and skip execution when --help flag is passed to any CLI command

## Description

If any CLI command is invoked with a `--help` flag, it should display the command's help text and NOT perform the command's action. This should apply globally to all commands, not just specific ones.

## Requirements

1. **Global --help interception** — before any command action runs, check for the presence of `--help` in the arguments. If present, display the command's help text and exit cleanly.

2. **No side effects** — when `--help` is passed, the command must not perform any of its normal operations (no file I/O, no API calls, no loop execution, etc.).

3. **All existing tests must continue to pass.**

## Produces

- Updated CLI argument parsing to intercept `--help` globally across all commands
- Help text displayed for each command when `--help` is passed
- Tests verifying `--help` shows help and does not execute the command
