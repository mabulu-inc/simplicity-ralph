# T-019: Externalize pricing and complexity tier configuration

- **Status**: TODO
- **Milestone**: 8 — Robustness
- **Depends**: none
- **PRD Reference**: §8.2

## Description

Move hardcoded operational parameters out of source code so they can be updated without rebuilding. Currently:

- `PRICE_INPUT`, `PRICE_CACHE_WRITE`, etc. are hardcoded in `commands/cost.ts`
- `TIER_SCALING` thresholds are hardcoded in `commands/loop.ts`

**Requirements:**

- Define sensible defaults in code (current values)
- Allow overrides via environment variables (e.g., `RALPH_PRICE_INPUT`) and/or a config file section
- Document the override mechanism

## Produces

- Updated `src/ralph/commands/cost.ts`
- Updated `src/ralph/commands/loop.ts`
- Config loading utility or extension to `src/ralph/core/config.ts`
- Tests for override behavior
