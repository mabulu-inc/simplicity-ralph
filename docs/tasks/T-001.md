## Description

Parse task files (`docs/tasks/T-NNN.md`) and extract structured metadata. Must handle the full task file format defined in the PRD.

**Capabilities:**

- Parse a single task file and return structured data (status, milestone, depends, PRD reference, completed, commit, cost, title, description)
- Scan a directory of task files and return all tasks
- Find the next eligible task (lowest-numbered TODO with all deps met and no `## Blocked` section)
- Count tasks by status
- Check if all tasks are done
