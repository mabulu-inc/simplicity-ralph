## Description

Implement `ralph loop` — the main AI development loop. This is ralph's core command.

**Iteration cycle:**

1. Pre-flight checks (claude CLI available, tasks directory exists)
2. Start database containers if project has docker-compose.yml
3. Discard unstaged changes from crashed iterations (clean slate)
4. Find next eligible task
5. Spawn `claude --print` with the boot prompt
6. Monitor progress (tool use, phases, errors) via JSON stream
7. Timeout — kill if iteration exceeds time limit
8. Commit detection — after a commit lands, end the iteration
9. Post-iteration: backfill SHAs, update costs, regenerate milestones, push

**Options:**

- `-n, --iterations <N>` — max iterations (default: 10, 0 = unlimited)
- `-d, --delay <seconds>` — delay between iterations (default: 2)
- `-t, --timeout <seconds>` — max per iteration (default: 900)
- `-v, --verbose` — stream output to terminal
- `--dry-run` — print config and exit
- `--no-push` — skip auto-push
- `--no-db` — skip database startup

**Boot prompt:** must encode the full methodology — TDD, quality gates, tool usage rules, one commit per task, task file update in same commit.
