## Description

Add a `--json` flag to `ralph monitor` that outputs the dashboard data as JSON instead of formatted text. This enables scripting and integration with external tools.

**Requirements:**

- Add `-j, --json` flag to the monitor command
- When `--json` is passed, output the `MonitorData` object as a single-line JSON string to stdout
- Include all fields: status, done, total, currentTaskId, currentTaskTitle, phases, currentPhaseStarted, lastLogLine
- In watch mode with `--json`, output one JSON line per refresh (JSONL format)
- All existing tests must continue to pass

## Hints

- The `MonitorData` interface already has all the fields — just serialize it
- `JSON.stringify(data)` is sufficient; dates should be ISO strings
- This is a small addition to the existing `run()` function

## Produces

- Updated `src/ralph/commands/monitor.ts`
- Tests for JSON output mode
