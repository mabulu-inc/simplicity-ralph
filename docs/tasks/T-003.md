## Description

Provide typed functions for the git operations ralph needs.

**Capabilities:**

- Get current HEAD SHA
- Check if working tree is clean (staged vs unstaged)
- Discard unstaged changes
- Get commit log (oneline format, with optional filtering)
- Find commit by message pattern (e.g., find "T-NNN:" commit)
- Check if there are unpushed commits vs a remote branch
- Push to remote
- Add files and commit

All operations should accept a `cwd` parameter so they work from any project directory.
