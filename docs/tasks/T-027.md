## Description

The monitor watch mode uses recursive `setInterval` calls and appends output, producing a streaming log rather than a live dashboard. Refactor to clear the screen before each refresh so the display updates in place.

**Requirements:**

- Replace the recursive self-calling pattern with a straightforward `setInterval` loop
- Clear the terminal before each refresh (e.g., `console.clear()` or ANSI escape `\x1Bc`)
- Handle clean shutdown (clear the interval on Ctrl+C / SIGINT)
- All existing tests must continue to pass

- Updated tests as needed
