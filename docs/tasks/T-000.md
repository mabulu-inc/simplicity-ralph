# T-000: Project infrastructure and CLI skeleton

- **Status**: TODO
- **Milestone**: 1 — Infrastructure
- **Depends**: none
- **PRD Reference**: §3

## Description

Set up the ralph CLI package infrastructure. Create the directory structure under `src/ralph/`, add the bin entry to `package.json`, set up husky pre-commit hooks, and create a minimal CLI entry point that dispatches subcommands.

The CLI should accept `ralph <command>` where command is one of: `init`, `loop`, `monitor`, `kill`, `milestones`, `shas`, `cost`. Unknown commands or no command should print usage help.

## Produces

- `src/ralph/cli.ts` — entry point with hashbang, arg parsing, subcommand dispatch
- `src/ralph/commands/` — directory with stub files for each command
- Husky pre-commit hook wired up
- Tests verifying CLI dispatches correctly and prints help
