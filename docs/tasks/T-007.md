# T-007: Milestones command

- **Status**: TODO
- **Milestone**: 3 — Utility Commands
- **Depends**: T-001
- **PRD Reference**: §3.5

## Description

Implement `ralph milestones` — generate `docs/MILESTONES.md` from task files.

**Behavior:**

- Scan all task files and extract milestone, status, title, cost
- Group tasks by milestone (in order of first appearance)
- Generate markdown with checkboxes, per-milestone cost rollup, and grand total
- Write to `docs/MILESTONES.md`

## Produces

- `src/ralph/commands/milestones.ts`
- Tests verifying correct grouping, cost rollup, and markdown output
