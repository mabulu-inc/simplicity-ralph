# T-004: Process management module

- **Status**: TODO
- **Milestone**: 2 — Core Modules
- **Depends**: T-000
- **PRD Reference**: §3.2, §3.4

## Description

Manage child processes for ralph — primarily spawning and monitoring Claude CLI sessions, but also handling process tree cleanup.

**Capabilities:**

- Spawn a child process and capture output to a file
- Kill a process tree (process + all descendants)
- Find processes by command pattern (for kill command)
- Monitor a running process (detect completion, timeout)

Process management must handle graceful shutdown (SIGTERM first, then SIGKILL after timeout) and proper cleanup on parent exit.

## Produces

- `src/ralph/core/process.ts`
- Tests covering: spawn and capture, process tree kill, timeout behavior
