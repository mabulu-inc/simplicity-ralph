# T-026: Migrate init.ts from sync to async file system APIs

- **Status**: TODO
- **Milestone**: 8 — Robustness
- **Depends**: none
- **PRD Reference**: §8.7

## Description

`init.ts` uses synchronous fs calls (`mkdirSync`, `writeFileSync`) while the rest of the codebase uses `node:fs/promises`. Migrate to async/await for consistency.

**Requirements:**

- Replace all synchronous fs calls in `init.ts` with their async equivalents from `node:fs/promises`
- No behavior changes — pure refactor
- All existing tests must continue to pass

## Produces

- Updated `src/ralph/commands/init.ts`
- Updated tests as needed
