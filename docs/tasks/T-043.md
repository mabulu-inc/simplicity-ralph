# T-043: Add project rules template and slim down agent instructions files

- **Status**: TODO
- **Milestone**: 10 — Prompt Quality & Scalability
- **Depends**: T-035
- **PRD Reference**: §5.9, §5.1, §5.7

## Description

Add `docs/prompts/rules.md` as the home for project-specific rules (e.g., "code goes under `src/ralph/`", "do not use TodoWrite"). These rules are currently duplicated in agent instructions files or missing entirely. This task makes them agent-agnostic and part of the prompt template system.

**Requirements:**

- Add `{{project.rules}}` template variable: at prompt build time, read `docs/prompts/rules.md` and inject its contents. If the file does not exist or is empty, resolve to empty string.
- Update `docs/prompts/boot.md` default template to include `{{project.rules}}` in an appropriate location (after PROJECT CONFIG, before WORKFLOW)
- Update `ralph init` to generate a default `docs/prompts/rules.md` with a comment explaining its purpose and example rules
- Slim down agent instructions file templates (`claude-md.ts`, `gemini-md.ts`, `agents-md.ts`, `cursor-rules.ts`, `continue-yaml.ts`): remove the `## Project-Specific Config` section — config lives in `ralph.config.json`, not duplicated in instructions files. Keep only project goal and methodology pointer.
- Update `loadAndInterpolate` (or equivalent) in `prompt-template.ts` to support the new `{{project.rules}}` variable
- All existing tests must continue to pass

## Hints

- The `loadAndInterpolate` function in `core/prompt-template.ts` already handles `{{config.*}}` and `{{task.*}}` — add `{{project.rules}}` alongside those
- Agent instructions files should become ~5 lines: project name, goal, methodology pointer
- The rules file is read at prompt build time, not at config parse time

## Produces

- `src/ralph/templates/rules-md.ts` — default rules template generator
- Updated `src/ralph/core/prompt-template.ts` — `{{project.rules}}` interpolation
- Updated `src/ralph/templates/claude-md.ts` — remove config section
- Updated `src/ralph/templates/gemini-md.ts` — remove config section
- Updated `src/ralph/templates/agents-md.ts` — remove config section
- Updated `src/ralph/templates/cursor-rules.ts` — remove config section
- Updated `src/ralph/templates/continue-yaml.ts` — remove config section
- Updated `src/ralph/commands/init.ts` — generate rules.md
- Updated `src/ralph/templates/boot-prompt.ts` — include `{{project.rules}}`
- Tests for rules loading and interpolation
