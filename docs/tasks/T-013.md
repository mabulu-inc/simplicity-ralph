## Description

End-to-end tests that verify ralph commands work together correctly.

**Test scenarios:**

- `ralph init` in a fresh directory creates expected file structure
- `ralph milestones` generates correct output from task files
- `ralph shas` backfills SHAs correctly
- `ralph cost` calculates costs from sample logs
- `ralph monitor` renders status correctly
- `ralph kill` must NOT be tested — it searches for and kills `claude` processes, which terminates the running Claude Code session
- CLI dispatches commands correctly and shows help for unknown commands
- `ralph loop --dry-run` shows auto-scaled complexity tier, turns, and timeout

These tests exercise the full command pipeline, not just individual modules.
