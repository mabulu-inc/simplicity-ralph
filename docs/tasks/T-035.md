# T-035: Update ralph init to scaffold ralph.config.json and prompt agent selection

- **Status**: TODO
- **Milestone**: 9 — Agent Abstraction
- **Depends**: T-031
- **PRD Reference**: §2, §3.1

## Description

Update the `ralph init` flow to produce a `ralph.config.json` file and prompt the user for their preferred AI coding agent.

**Requirements:**

- Add an "AI agent" prompt to the init flow (choices: claude, gemini, codex, continue, cursor — default: claude)
- Generate `ralph.config.json` at the project root containing all config fields (language, package manager, test framework, quality check, test command, agent, etc.)
- Update config reader (`core/config.ts`) to read from `ralph.config.json` instead of (or in addition to) extracting fields from `.claude/CLAUDE.md`
- Still generate the agent-specific instructions file (e.g., `.claude/CLAUDE.md`, `GEMINI.md`) for the selected agent, but ralph's own config lives in `ralph.config.json`
- All existing tests must be updated

## Produces

- Updated `src/ralph/commands/init.ts`
- Updated `src/ralph/core/config.ts`
- Updated `src/ralph/templates/` (new template for ralph.config.json)
- Tests for config reading from ralph.config.json
