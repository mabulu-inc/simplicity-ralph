# T-060: Monitor picks wrong log file when task IDs are non-sequential

## Description

`findLatestLogFile()` in `src/ralph/commands/monitor.ts` sorts log filenames alphabetically and picks the last one. This breaks when the loop processes tasks out of numeric order — e.g., T-069 after T-088 — because `T-088-*.jsonl` sorts after `T-069-*.jsonl` alphabetically, so the monitor shows the stale T-088 log instead of the active T-069 log.

## Requirements

1. **Fix `findLatestLogFile()`** — sort log files by filesystem modification time (newest first) instead of alphabetically by name.

2. **All existing tests must continue to pass.**

## Produces

- Updated `findLatestLogFile()` in `src/ralph/commands/monitor.ts`
- Test coverage for the fix
