## Description

Implement the Continue CLI and Cursor CLI providers using the agent provider interface from T-031. These share nearly identical CLI conventions with Claude Code.

**Requirements:**

Continue CLI:

- Binary: `cn`
- Print mode: `-p`
- JSON output: `--output-format stream-json`
- Max turns: `--max-turns N` (supported)
- Instructions file: `~/.continue/config.yaml`

Cursor CLI:

- Binary: `cursor`
- Print mode: `-p`
- JSON output: `--output-format stream-json`
- Max turns: not supported — rely on ralph timeout
- Instructions file: `.cursor/rules/`

For both:

- Parse NDJSON stream into ralph's normalized event format
- Update `ralph init` to generate the appropriate instructions file when selected

## Produces

- `src/ralph/providers/continue.ts`
- `src/ralph/providers/cursor.ts`
- Updated `src/ralph/commands/init.ts`
- Tests for both providers
