## Description

The README and the Astro Starlight docs site (`docs-site/`) reference the old prompt architecture where `ralph init` copies template files into the user's project and `ralph update` refreshes them. After the built-in-first refactor (T-076 through T-079), these references are stale and misleading.

The docs site is an Astro Starlight project at `docs-site/` with content in `docs-site/src/content/docs/`. It has sections for Getting Started, Core Concepts, Commands, Agents, and Guides.
