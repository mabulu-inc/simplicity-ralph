# T-002: Project config parser

- **Status**: TODO
- **Milestone**: 2 — Core Modules
- **Depends**: T-000
- **PRD Reference**: §2, §2.1, §2.2

## Description

Read project configuration from `.claude/CLAUDE.md`. Extract the structured config fields that ralph needs to operate.

**Capabilities:**

- Read a CLAUDE.md file and extract: language, package manager, testing framework, quality check command, test command
- Handle optional fields: database config, file naming convention
- Return a typed config object
- Validate that required fields are present (check command is mandatory for loop)

## Produces

- `src/ralph/core/config.ts`
- Tests covering: full config extraction, missing fields, optional fields, validation
