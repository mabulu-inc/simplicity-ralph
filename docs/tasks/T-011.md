## Description

Create embedded templates for the `ralph init` command. These are TypeScript functions that generate file content based on project configuration.

**Templates needed:**

- **CLAUDE.md** — project config with methodology reference, filled with user's answers
- **RALPH-METHODOLOGY.md** — full methodology reference document
- **PRD.md** — skeleton PRD with numbered sections
- **T-000.md** — infrastructure bootstrap task, customized for the project's stack

Each template is a function that takes config parameters and returns the file content as a string.

- `src/ralph/templates/claude-md.ts`
- `src/ralph/templates/methodology.ts`
- `src/ralph/templates/prd.ts`

- Tests verifying templates produce valid content for different project configs
