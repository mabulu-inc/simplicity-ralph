## Description

Implement `ralph monitor` — real-time status display.

**Displays:**

- Ralph status: RUNNING / BETWEEN TASKS / STOPPED
- Progress bar with task counts (done/total, percentage)
- Current task ID and title (from most recent log file)
- Phase timeline for the active iteration (from `[PHASE]` markers in log)

**Options:**

- Default: one-shot status display
- `-w, --watch` — continuous mode, refresh on interval
- `-i, --interval <seconds>` — refresh interval (default: 5)
