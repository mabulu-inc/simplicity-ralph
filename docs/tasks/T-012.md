## Description

Implement `ralph init` — interactive project bootstrapper.

**Prompts:**

1. Project name
2. Language (TypeScript, Python, Go, Rust, etc.)
3. Package manager (pnpm, npm, yarn, pip, cargo, etc.)
4. Test framework (Vitest, Jest, pytest, etc.)
5. Check command — required, helps configure if missing
6. Database (none, PostgreSQL via Docker, etc.)

**Creates (using templates from T-011):**

- `docs/PRD.md` — skeleton PRD
- `docs/RALPH-METHODOLOGY.md` — methodology reference
- `docs/tasks/T-000.md` — bootstrap task
- `.claude/CLAUDE.md` — project config

**Behavior:**

- If files already exist, warn and ask before overwriting
- For Node.js projects, add ralph scripts to `package.json`
- If no check command exists, scaffold one
- Print summary of what was created

- `src/ralph/commands/init.ts`
- Tests (can test non-interactive mode with pre-supplied answers)
