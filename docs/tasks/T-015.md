## Description

Create a GitHub Actions workflow that publishes the package to npm when a version tag (`v*`) is pushed.

**Behavior:**

- Triggered on push of tags matching `v*`
- Install pnpm and Node.js 20 with npm registry configured
- Run `pnpm install --frozen-lockfile`
- Run `pnpm check` to verify quality before publishing
- Run `pnpm publish --access public --no-git-checks`
- Uses `NPM_TOKEN` secret for authentication

## Produces

- `.github/workflows/publish.yml`
- Tests verifying the workflow file is valid YAML with expected structure
