# T-014: CI workflow

- **Status**: TODO
- **Milestone**: 7 — CI/CD & Publishing
- **Depends**: T-000
- **PRD Reference**: §6

## Description

Create a GitHub Actions CI workflow that runs on push to `main` and on pull requests.

**Behavior:**

- Install pnpm and Node.js 20
- Run `pnpm install --frozen-lockfile`
- Run `pnpm check` (lint → format → typecheck → build → test:coverage)

## Produces

- `.github/workflows/ci.yml`
- Tests verifying the workflow file is valid YAML with expected structure
